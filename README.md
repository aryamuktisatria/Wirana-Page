Ikuti langkah-langkah sederhana ini untuk menjalankan proyek secara lokal:

1.  **Clone Repositori:**
    ```bash
    git clone [https://github.com/aryamuktisatria/Wirana-Page.git](https://github.com/aryamuktisatria/Wirana-Page.git)
    ```
